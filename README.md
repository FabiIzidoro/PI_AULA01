# CRUD de Produtos com Node.js e Express

> Atividade de implementação de CRUD para gerenciamento de produtos.

Este projeto é uma API simples desenvolvida com **Node.js** e **Express** que permite realizar operações CRUD (Criar, Ler, Atualizar e Excluir) para gerenciamento de produtos. Ele utiliza uma estrutura de dados em memória para armazenar os produtos, com as seguintes informações:

- **id**: Identificador único do produto.
- **nome**: Nome do produto.
- **preço**: Preço do produto.

## Funcionalidades Principais
- **Cadastrar um novo produto.**
- **Listar todos os produtos cadastrados.**
- **Buscar um produto específico por ID.**
- **Atualizar todos os dados de um produto.**
- **Atualizar parcialmente os dados de um produto.**
- **Excluir um produto.**

## Tecnologias

- **Node.js**: Ambiente de execução JavaScript no servidor.
- **Express**: Framework para desenvolvimento de APIs.
- **Postman**: Para testar as rotas da API.
